# Изображения дашбордов для страницы "О нас"

## Требуемые изображения

Для правильного отображения дашбордов на странице "О нас" необходимо добавить три изображения:

1. `dashboard_1.png` - Первый (передний) дашборд
2. `dashboard_2.png` - Второй (средний) дашборд  
3. `dashboard_3.png` - Третий (задний) дашборд

## Как получить изображения из Figma

1. Откройте макет страницы "О нас" в Figma
2. Найдите три дашборда справа на странице
3. Выделите каждый дашборд и экспортируйте как PNG:
   - Выделите дашборд
   - В правой панели найдите раздел "Export"
   - Выберите формат PNG, масштаб 2x
   - Экспортируйте и сохраните как `dashboard_1.png`, `dashboard_2.png`, `dashboard_3.png`
4. Поместите все три файла в папку `public/dashboards/`

## Альтернативный способ (через скрипт)

Запустите скрипт `download_dashboard_images.py` для автоматической загрузки изображений из Figma:

```bash
python download_dashboard_images.py
```

**Примечание:** Перед запуском скрипта убедитесь, что у вас есть правильный node ID для страницы "О нас" в Figma.

## Fallback

Если изображения `dashboard_1.png`, `dashboard_2.png`, `dashboard_3.png` не найдены, компонент автоматически использует существующее изображение `f5c8294ae51b66f97c287dbc485c071ba533518c.png` для всех трех дашбордов.
